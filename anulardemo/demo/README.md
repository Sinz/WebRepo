## TodoMVC 页面开发 

**1. 需求说明**

1.1 数据列表展示

- 有数据 
- 无数据 

 使用 *ngFor / *ngIF 结合ng-template标签 , 循环 和 条件判断

1.2 添加任务

- 页面初始化的时候文本框获得焦点
- 敲击回车添加到任务列表中
- 不允许有非空数据
- 添加完成清空文本框

1.3 标记所有任务完成/未完成

1.4 任务项功能

- 切换任务完成状态
    - 样式联动
    - 联动切换所有任务状态
- 删除任务项
- 双击laber进入编辑模式

1.5 编辑任务项

- 编辑文本框自动获得焦点
- 在文本框中敲击回车失去焦点
- 输入状态下esc 取消编辑

1.6 显示所有未完成任务数

1.7 清楚所有已完成任务

1.8 将数据持久化到localStorage中

1.9 路由状态切换

- 点击连接过滤数据的输出
- 刷新保持过滤状态
- 切换点击链接的样式
